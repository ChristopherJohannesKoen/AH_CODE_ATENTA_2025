# Code of Conduct

## Our Standard

We are committed to a respectful, inclusive, and constructive collaboration environment.

## Expected Behavior

- Be respectful and professional.
- Assume good intent and ask clarifying questions.
- Give technical feedback that is specific and actionable.
- Focus on improving code and outcomes, not individuals.

## Unacceptable Behavior

- Harassment, abuse, or discrimination.
- Personal attacks or inflammatory language.
- Publishing private information without consent.
- Repeatedly derailing technical discussions.

## Enforcement

Maintainers may remove or moderate comments, issues, or contributions that violate this code of conduct.

## Reporting

Report unacceptable behavior privately to maintainers. Include links, screenshots, and context where possible.
